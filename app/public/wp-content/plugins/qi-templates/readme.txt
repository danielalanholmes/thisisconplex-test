Qi Templates is templates for websites in all kinds of niches - from business to fashion, from tech to wedding. Developed by Qode Interactive.

== Installation ==

= Minimum Requirements =

* WordPress 5.8 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.2 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 128 MB or greater

= Installation =

From your WordPress dashboard

1. Download plugin from qodeinteractive.com
1. Go to your WordPress Dashboard -> Plugins -> Add New -> Upload Plugin
2. Upload Qi Templates
3. Activate Qi Templates from your Plugins page.
4. Enjoy :)
